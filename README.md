## Overview

ff-item is a static web database built for browsing Free Fire content in one place. It focuses on game items such as skins, bundles, emotes, characters, pets, cosmetics, and loot crate entries, all accessible through a straightforward browser interface.

Use it when you need item information without jumping between separate pages or buried menus. The search and filtering tools help you narrow results quickly, while category navigation makes it easier to move across item types and open individual records.
